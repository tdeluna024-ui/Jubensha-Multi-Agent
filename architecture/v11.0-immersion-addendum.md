# v11.0 沉浸与推理深度加层

> v11.0 = v9.1.1 基础 + 沉浸/推理深度补丁
>
> 触发：真人玩家对 v9.1.1 产出的《沪滩残局》给出"几乎没有可推、无反转、第一幕就猜 80%"的反馈
> 关系：与 v10（多媒体加层）正交独立。未来可合并为 v12 = v11 沉浸 + v10 漫画
> 状态：v11.0 / 2026-06-06

诊断详见 `_v11.0_player_feedback_diagnosis.md`。本文是架构层的正式描述。

---

## 0. v11.0 的核心命题

v9.1.1 的所有 agent 都在保证「**流程跑通**」：诡计能解出来、线索齐全、文本通顺、逻辑自洽、玩家能推到真相。

v9.1.1 缺一个**对偶维度**：「**沉浸够深**」。即——诡计够难（不秒破）、线索够阴（不直给）、文本够深（玩家共情）、反转够多（不一眼看穿）、节奏够紧（不流水账）。

**E-02 盲测的评分维度甚至反向奖励了简单本**：

> "凶手可识别度"满分 = 5 个玩家都推到 = 但这恰恰是玩家 1 分钟就能推到的标志

v11.0 = 把"沉浸"提升为一等公民，与"正确性"双轴并重。

---

## 1. 第 8 条法则（v11.0 新增）

> **法则八：流程跑通是底线，沉浸够深是目标。任何一项 QA 评分都必须有「难度/深度」对偶维度。**

具体而言：

- 凡是奖励"玩家能做到 X"的评分维度，必须配一个奖励"玩家做不到 X"的对偶维度
- 凡是 must_reveal 强制的，必须配 must_hide_until 强制（不允许提前泄底）
- 凡是有"线索数量"的，必须有"线索深度"（直白线索不计入）
- 凡是有"字数下限"的，必须有"情绪锚点数下限"（凑字数不算数）

## 3. 新增 agent：B-05 沉浸设计师

**部门**：B. 策划部
**上游**：B-02 悖论架构师 / B-03 结构设计师
**下游**：C 部门 + D 部门 + E 部门（全员读 immersion_blueprint）
**完整规格**：`02_agents/B-05-immersion-architect.md`

**职责**：把 B-02 的诡计骨架 + B-03 的幕结构升级为**沉浸蓝图**：
- 反转点
- 张力曲线
- 共情锚点
- 卡点（玩家必须暂停讨论的位置）
- 推理深度曲线（depth_targets，每幕玩家应能推到多少 %）

**关键输出**：`01_planning/immersion_blueprint.json`

```json
{
  "depth_targets": {
    "act_1_solvable_pct": 0.10,
    "act_2_solvable_pct": 0.30,
    "act_3_solvable_pct": 0.60,
    "act_4_solvable_pct": 0.85,
    "act_5_solvable_pct": 1.00
  },
  "reversal_points": [
    { "act": 2, "type": "凶手指向反转", "trigger_clue": "c_032" },
    { "act": 3, "type": "动机反转",     "trigger_clue": "c_041" }
  ],
  "tension_curve": [
    { "act": 1, "target_tension": 4 },
    { "act": 3, "target_tension": 9 }
  ],
  "empathy_anchors": [
    { "char": "char_01", "act": 2,
      "moment": "找到亡妻信件", "expected_emotion": "心疼" }
  ],
  "stuck_points": [
    { "act": 2, "phase": "中段", "duration_min": 15,
      "intended_block": "线索 c_001 和 c_007 同时成立但相矛盾" }
  ]
}
```

`depth_targets` 是整套 v11 的**灵魂字段**。沪滩残局的实际值是 act_1=80%，目标应该是 ≤10%。E-02 据此重新计算评分。

## 6. 四道质量门

升级后整本必须过四道门：

```
门 1 - B-04 锁 Bible 时
       要求：immersion_blueprint.json 已生成 + B-05 自检 ≥7.5

门 2 - C-04 数据分发后
       要求：clues.inference_depth ≥1 占 ≥50%
            details 三层埋雷比例 ≥30%/40%/30%（表/中/深）

门 3 - E-01 完工时
       要求：8 维度总分 ≥8.0
            其中"诡计深度" + "推理深度" 每项 ≥7

门 4 - E-02 完工时
       要求：推理曲线匹配度 ≥8
            act_1 平均把握度 ≤15%
            玩家共情度 ≥7
```

任一门 fail = 强制回炉。不允许"凑合通过"。
