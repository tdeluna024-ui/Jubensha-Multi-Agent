# 评分规则（Scoring Rubric）· v11.0 重构版

> v11.0 全面重构。所有 QA agent（E-01, E-02, B-05 自检）必须按本文档输出评分。
>
> **核心变化**：v9.1.1 的评分体系奖励"流程跑通"，v11.0 加上"沉浸够深"维度。E-02 的"凶手可识别度"被删除，因为它反向奖励了简单本（沪滩残局事故）。

---

## 1. 10 分加权制（不变）

每个 QA agent 评估若干维度，每个维度：
- 评分 0-10（一位小数）
- 加权重（所有维度权重和 = 1.0）
- 总分 = 加权得分之和（范围 0-10）

## 2. E-01 逻辑审计：8 维度

| 维度 | 默认权重 | 评判方式 |
|------|---------|---------|
| 跨角色一致性 | 0.20 | 比对所有 final，找事件三元组冲突 |
| 时间线一致性 | 0.15 | 同一角色不同时刻位置/动作冲突 |
| 诡计自洽性 | 0.10 | 检查 paradox 的物理约束是否被违反 |

| **推理深度** ⭐ v11 | **0.15** | clues.inference_depth ≥1 占比 / details 三层埋雷比例 |

### 评分细则（v11 新增维度）

#### 诡计深度（0.15）
| 分数 | 标准 |

| 10 | ≥3 层真相 + 每层 ≥2 个 key_distinguishing_clues + 暗凶 |
| 8 | 2 层真相 + 区分线索齐备 |
| 6 | 1 层真相但有 ≥3 个反转铺垫 |
| 3 | 1 层真相 + 明凶 |
| 0 | 无诡计（沪滩残局 v1 = 0） |

#### 反转点存在度（0.10）
| 分数 | 标准 |
|------|------|
| 10 | ≥3 个反转 + 全部在 act_1 ¾ 之后 |
| 8 | 2 个反转 + 都在 act_1 ¾ 之后 |
| 6 | 2 个反转但有 1 个在 act_1 |
| 3 | 1 个反转 |
| 0 | 0 反转（沪滩残局 v1 = 0） |

#### 推理深度（0.15）
| 分数 | 标准 |
|------|------|
| 10 | clues.inference_depth ≥1 占 ≥70% + details 三层埋雷比 30/40/30 |
| 8 | inference_depth ≥1 占 ≥50% + 三层埋雷比 25/45/30 |
| 6 | inference_depth ≥1 占 ≥30% |
| 3 | inference_depth ≥1 占 ≥10% |
| 0 | 全是直白线索（沪滩残局 v1 ≈ 0） |

#### 卡点设计（0.10）
| 分数 | 标准 |
|------|------|
| 10 | ≥3 个卡点 + 全部有 resolution_clue + 时长分布均匀 |
| 8 | ≥3 个卡点 + 全部有 resolution_clue |
| 6 | 2 个卡点 + 完备 |
| 3 | 1 个卡点 |
| 0 | 0 卡点（沪滩残局 v1 = 0） |

### pass_threshold

- 商业本：8.5
- demo 本：7.0
- 单维度最低：**诡计深度 ≥7 + 推理深度 ≥7**（这两项不达标整体不通过）

## 3. E-02 盲测：7 维度（核心反转）

| 维度 | 默认权重 | 评判方式 |
|------|---------|---------|
| **推理曲线匹配度** ⭐ v11 替代 | **0.30** | 对比 depth_targets：subagent 每幕"猜中把握度" |
| 动机清晰度 | 0.15 | 仅在最后一幕评：subagent 是否推对动机 |
| 信息分发均衡度 | 0.10 | subagent 间信息密度方差 |
| 无上帝视角断点 | 0.10 | 是否存在超出可见信息的逻辑跳跃 |
| **反转触发度** ⭐ v11 | **0.15** | subagent 是否在反转点真的"惊讶" |
| **玩家共情度** ⭐ v11 | **0.15** | subagent 在 empathy_anchor 位置是否表达对应情绪 |
| **卡点真实度** ⭐ v11 | **0.05** | subagent 在 stuck_point 位置是否真的卡住 |

### v9.1.1 → v11.0 关键差异

```
v9.1.1: 凶手可识别度（权重 0.4）
        → 5/5 推到 = 满分
        → 但沪滩残局是 5/5 推到 = 满分 ❌

v11.0: 推理曲线匹配度（权重 0.3）
       → 对比 depth_targets
       → act_1 把握度 80%（沪滩残局）→ 严重失败
       → act_1 把握度 10% + act_5 把握度 95% → 满分
```

### 推理曲线匹配度评分细则

让每个 subagent 在每幕结束输出 `confidence_pct_on_killer`（0-100）。

计算每幕的 N 个 subagent 平均把握度：
```
avg_confidence[act] = mean(subagent.confidence[act])
```

对比 depth_targets：

```
deviation[act] = |avg_confidence[act] / 100 - depth_targets["act_K_solvable_pct"]|
```

总偏差 = sum(deviation) / total_acts

| 总偏差 | 分数 |
|--------|------|
| < 0.05 | 10 |
| 0.05-0.10 | 9 |
| 0.10-0.15 | 8 |
| 0.15-0.20 | 6 |
| 0.20-0.30 | 4 |
| 0.30-0.50 | 2 |
| > 0.50 | 0 |

**沪滩残局 v1 实测**：
- target act_1 = 0.10, actual = 0.80
- deviation = 0.70
- → 0 分

### 反转触发度评分细则

在每个 reversal_point 位置（来自 immersion_blueprint），让 subagent 描述"这一幕我的推理变化"。

| 标准 | 分数 |
|------|------|
| subagent 明确说"我之前的推理被推翻了，现在转向 X" | 10 |
| subagent 说"我有点动摇，但仍然倾向之前的判断" | 6 |
| subagent 完全没察觉反转 | 2 |
| 反转点不存在或 subagent 一开始就知道真相 | 0 |

### 玩家共情度评分细则

在每个 empathy_anchor 位置，让 subagent 写"我此刻的情绪"。

| 标准 | 分数 |
|------|------|
| subagent 用了 expected_emotion 词或近义词 | 10 |
| subagent 表达了相关情绪（如该共情却说"困惑"） | 6 |
| subagent 无情绪表达 | 3 |
| subagent 表达了相反情绪 | 0 |

### pass_threshold
- 默认：**8.0**
- 单维度最低：**推理曲线匹配度 ≥7**（这项不达标整体不通过）

## 4. B-05 沉浸蓝图自检（新增）

不是 QA agent，是 B-05 自己的输出门。

| 自检项 | 通过标准 |
|--------|---------|
| act_1 depth | depth_targets.act_1 ≤ 0.20（深度本 ≤ 0.10） |
| reversal count | reversal_points 长度 ≥ 2 |
| reversal timing | 所有 reversal_points 都晚于 act_1 ¾ |
| empathy coverage | 每个关键角色 ≥1 个共情锚 |
| stuck points | stuck_points ≥3 + 全有 resolution_clue |

5 项必须全 PASS 才能写盘锁版。

总分（用于 trace 记录）：每项 2 分，全 PASS = 10。

## 7. Blockers 严重度（不变）

| Severity | 标准 |
|----------|------|
| critical | 不修无法发版（如 v11 默认: 推理曲线匹配度 = 0） |
| high | 严重影响推理可达性 / 重大逻辑漏洞 |
| medium | 影响体验但可推理 |
| low | 美化级别（可推迟） |

## 8. 反例（v11 加强）

- ❌ 给"凶手可识别度"高分（v11 不存在这个维度）
- ❌ "推理曲线匹配度"评判依据 = "我觉得差不多"（必须是 subagent 实际把握度的数学计算）
- ❌ blocker 无 root_cause_agent + fix_proposal（无法迭代）
- ❌ 推理深度 = 0 但整体仍 pass（必须单维度最低门限）
- ❌ 沪滩残局这样的"5/5 推到"还给 8 分
